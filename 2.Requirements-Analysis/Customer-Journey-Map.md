# Customer Journey Map

## 1. Enter

### Step

Open Service Portal

### Goal

Submit an employee onboarding/offboarding request.

### Interaction

* Login to ServiceNow

* Open Service Portal

* Search Employee Onboarding & Offboarding

### Positive

Easy access through Service Catalog.

### Negative

User may not know where the catalog item is.

### Opportunity

Improve search and categorize HR services.

## 2. Browse

### Step

Search for Employee Onboarding & Offboarding

### Goal

Locate and select the correct catalog item.

## 3. Engage

### Step

Open Employee Onboarding & Offboarding Catalog Item

### Goal

Fill employee details.

### Interaction

* Employee ID

* Employee Name

* Department

* Manager

* Request Type

* Joining Date / Exit Date

### Positive

Simple and user-friendly form.

### Negative

Missing mandatory fields stop submission.

### Opportunity

Auto-fill department and manager information.

## 4. Submit Request

### Step

Submit the catalog request.

### Goal

Create onboarding request.

### Interaction

Click **Order Now / Checkout**.

### Positive

Request number generated instantly.

### Negative

User cannot immediately see backend processing.

### Opportunity

Display request progress in real time.

## 5. Manager Approval

### Step

Manager receives approval request.

### Goal

Approve or reject.

### Interaction

Approval record created.

### Positive

Automated approval workflow.

### Negative

Delay if manager doesn't respond.

### Opportunity

Reminder email notifications.

## 6. IT + Facilities + Security Tasks

### Step

Flow Designer creates departmental tasks.

### Interaction

Tasks created for:

* IT Team

* Facilities Team

* Security Team

### Positive

No manual task assignment.

### Negative

Delay if department ignores task.

### Opportunity

SLA monitoring and escalation.

## 7. Request Completed

### Step

Departments complete assigned tasks.

### Interaction

* IT account created

* Workstation prepared

* Building/System access provided

### Positive

Parallel task execution saves time.

### Negative

One delayed department delays onboarding.

### Opportunity

Dashboard monitoring.

## 8. Dashboard & Reports

### Step

HR reviews reports.

### Interaction

* Dashboard

* Reports

* SLA

### Positive

Easy monitoring.

### Negative

Need future HRMS integration.

### Opportunity

* Active Directory Integration

* HRMS Integration

* Asset Management Integration
