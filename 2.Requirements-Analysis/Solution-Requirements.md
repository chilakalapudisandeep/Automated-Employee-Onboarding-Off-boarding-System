# Solution Requirements

# Project Name

**Automated Employee Onboarding & Offboarding System**

# Functional Requirements

| FR No | Functional Requirement (Epic) | Sub Requirement (Story / Sub-Task) |
| --- | --- | --- |
| FR-1 | Employee Onboarding Request | HR submits onboarding request through Service Catalog |
| FR-2 | Employee Offboarding Request | HR submits offboarding request through Service Catalog |
| FR-3 | Manager Approval | Route request to the employee's manager for approval |
| FR-4 | Employee Lifecycle Record | Create a record in the custom Employee Lifecycle Management table |
| FR-5 | IT Task Automation | Automatically create an IT catalog task after approval |
| FR-6 | Facilities Task Automation | Automatically create a Facilities catalog task after approval |
| FR-7 | Security Task Automation | Automatically create a Security catalog task after approval |
| FR-8 | Email Notifications | Send notifications for approvals and request updates |
| FR-9 | SLA Tracking | Track task completion using SLA |
| FR-10 | Reports & Dashboard | Display request status and progress using reports and dashboards |

# Non-Functional Requirements

| NFR No | Non-Functional Requirement | Description |
| --- | --- | --- |
| NFR-1 | Usability | Simple Service Portal interface for HR users |
| NFR-2 | Security | Role-based ACLs for HR, IT, Facilities, Security, and Managers |
| NFR-3 | Reliability | Automated workflow execution with minimal manual intervention |
| NFR-4 | Performance | Request creation, approvals, and task generation should execute efficiently |
| NFR-5 | Availability | Accessible through the ServiceNow platform whenever required |
| NFR-6 | Scalability | Can be extended to support additional HR lifecycle services and integrations |
